## Automatização - dados de trânsito do Infosiga (SP)
#### Script para download e preparação automática dos dados do InfoSiga SP
#### Secretaria de Tecnologia, Inovação e Desenvolvimento Econômico
#### Osasco/SP
#### Data: Janeiro/2026

#### Esse script faz o download dos dados sobre segurança de trânsito direto do link oficial do governo (https://infosiga.detran.sp.gov.br/#referencia), transforma os dados conforme necessidade e sobe os bancos de dados para a pasta desejada.
#### O ChatGPT foi utilizado parcialmente para o desenvolvimento desse script.
